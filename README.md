# Meu-Kanban

Feito para uso profissional, utilizando Google ai studio, com intuito de auxiliar no acompanhamento e rastreabilidade de atividades.

Funciona offline, sendo somente um arquivo index.html, para que seja de simples execução, os dados são salvos localmente na maquina do usuario para que não tenha nenhum vinculo com a web, assim tornando a aplicação totalmente offline e portavel.
